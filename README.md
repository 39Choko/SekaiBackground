# Sekai Background

## Installation

```ps1
iwr -useb https://raw.githubusercontent.com/39Choko/SekaiBackground/master/install.ps1 | iex
```

then restart your terminal and type `sekai-bg` to start using it!

when you start for the first time, a config file is created in `C:\39Choko\SekaiBackground\config.json` where you can toggle a character
